# Columbia-Asia-Hospital-Data-Analysis
This project focuses on analyzing hospital revenue data to identify key income sources, optimize departmental performance, and develop strategies for improving patient affordability without compromising profitability.

Key Insights:

Analyzed revenue trends to pinpoint major income streams and areas for financial optimization.

Identified high-demand departments to support better staffing and resource allocation decisions.

Designed data-driven discount strategies to balance patient affordability and hospital profitability.

Tools & Technologies:

Power BI – for building interactive dashboards and visual insights.

Advanced Excel – for data cleaning, KPI tracking, and trend analysis.

SQL – for data extraction, transformation, and aggregation from relational databases.

Outcome:
Delivered actionable insights that helped optimize revenue distribution, enhance department efficiency, and improve financial decision-making across the hospital.
